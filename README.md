Project Title

GBDA302 Week 5: Reflective Camera Experience

---

Authors

Iba Abbasi

---

Description

This p5.js project demonstrates a reflective camera experience within a world that extends beyond the visible screen. The player moves slowly through a large environment while the camera follows using smooth eased motion instead of snapping directly to the player.

The project emphasizes pacing, atmosphere, and spatial awareness rather than challenge or competition. The camera is constrained to the world boundaries so it never reveals empty space outside the environment. Hidden interactive symbols are placed throughout the world and become visible as the player approaches them, encouraging quiet exploration.

The experience focuses on how motion, scale, and camera behavior alone can evoke emotion in interactive design.

---

Learning Goals

- Understand how to implement a scrolling camera system in a world larger than the viewport

- Constrain camera movement to prevent showing areas outside the world bounds

- Use easing (lerp) to create smooth and meditative camera motion

- Separate update and rendering logic into structured functions

- Design an experience where pacing and motion communicate emotion

- Implement hidden interactive elements triggered by player proximity
